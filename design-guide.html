Geometric Precision

This design system embodies a sharp, architectural aesthetic inspired by Brazilian modernism. Every element speaks to efficiency, professionalism, and corporate sophistication.

The zero border radius constraint creates a distinctive, memorable identity—sharp corners that cut through the noise of rounded, generic travel websites.

Animation Principles

Sharp, precise animations that feel efficient and professional. No bouncing, no elastic easing—just crisp, geometric motion.

Motion Guidelines

DURATION

Fast and snappy: 200-400ms for most interactions
Use 600-800ms only for page-level transitions

EASING

cubic-bezier(0.4, 0, 0.2, 1) - Sharp acceleration
Avoid elastic, bounce, or overly smooth curves

DIRECTION

Slide transitions (horizontal/vertical)
Geometric reveals with clip-path
Transform-based (translate, not position)

Example Animations

/* Slide In */
@keyframes slideInRight {
  from {
    transform: translateX(100px);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

/* Geometric Reveal */
@keyframes geometricReveal {
  from {
    clip-path: polygon(0 0, 0 0, 0 100%, 0 100%);
  }
  to {
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
  }
}

/* Button Hover */
.btn:hover {
  transform: translate(-4px, -4px);
  box-shadow: 12px 12px 0px 0px rgba(0,0,0,0.3);
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

Responsive Design

Mobile-first approach with thoughtful breakpoint usage and touch-friendly interactions.

Breakpoints

SM 640px
MD 768px

Mobile Considerations

  • ▪ Touch targets minimum 44x44px
  • ▪ Stack cards vertically on mobile
  • ▪ Collapse navigation to hamburger menu
  • ▪ Reduce type scale by 20-30% on mobile
  • ▪ Simplify shadows and animations
  • ▪ Maintain zero border radius across all devices
